# pm-toolkit

产品经理实用技能包：简历评审、NDA（保密协议）起草、隐私政策生成、语法与行文检查。产品工作之外不可或缺的辅助工具集。

## 技能列表（4 项）

- **draft-nda** — 起草两方之间详细的保密协议（NDA）。
- **grammar-check** — 识别文本中的语法、逻辑和行文问题，提供针对性修改建议，不对全文进行改写。
- **privacy-policy** — 为产品起草详细的隐私政策，涵盖数据类型、司法管辖区、合规考量及需法律审查的条款。
- **review-resume** — 基于 10 项最佳实践对产品经理简历进行全面评审与优化，包括 XYZ+S 公式、关键词优化、岗位定向适配和结构调整。

## 命令列表（5 条）

- `/pm-toolkit:draft-nda` — 起草两方之间的保密协议（NDA），包含适用管辖区的相关条款。
- `/pm-toolkit:privacy-policy` — 起草隐私政策，涵盖数据收集、使用、存储及合规要求。
- `/pm-toolkit:proofread` — 检查任意文本的语法、逻辑与行文——提供针对性修改，不全文改写。
- `/pm-toolkit:review-resume` — 基于 10 项最佳实践对产品经理简历进行全面评审——结构、量化指标、关键词及可落地的改进建议。
- `/pm-toolkit:tailor-resume` — 将产品经理简历针对特定职位描述进行定向优化——关键词对齐、经历重新定位与策略性调整。
