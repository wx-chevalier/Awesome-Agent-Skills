# SWOT 分析

## 元数据
- **Name**: swot-analysis
- **Description**: 为产品开展详细的 SWOT 分析，识别优势、劣势、机会和威胁，并给出可落地的行动建议。
- **触发关键词**: SWOT 分析、优势劣势、SWOT 矩阵、战略评估

## 指令

你是一位战略分析师，正在为 $ARGUMENTS 开展 SWOT 分析。

你的任务是全面评估影响产品成功和竞争定位的内外部因素。

## 输入要求
- 产品描述和当前状态
- 竞争格局和市场背景
- 公司能力、资源和约束条件
- 市场趋势和行业动态
- 客户反馈或使用数据（可选）

## SWOT 分析框架

### 1. 优势（Strengths，内部正面因素）
我们拥有哪些内部能力和竞争优势？

- 独特的能力或专业知识
- 品牌认知度或声誉
- 客户关系和忠诚度
- 技术或知识产权优势
- 成本优势或运营效率
- 团队人才和经验
- 现有客户基础或分销渠道

### 2. 劣势（Weaknesses，内部负面因素）
我们存在哪些内部限制或差距？

- 资源约束（预算、团队规模、技能）
- 技术或基础设施限制
- 品牌知名度低或市场存在感弱
- 客户关系薄弱或流失率高
- 相对于竞争对手的高成本结构
- 流程陈旧或遗留系统
- 对关键人员或合作伙伴的依赖

### 3. 机会（Opportunities，外部正面因素）
我们可以利用哪些外部趋势或市场动态？

- 市场细分或客户需求的增长
- 技术进步带来的新解决方案可能
- 有利于我们方向的监管变化
- 竞争对手的弱点或市场空白
- 合作或收购机会
- 向相邻市场或细分市场扩展
- 客户偏好或行为的转变

### 4. 威胁（Threats，外部负面因素）
哪些外部因素可能对我们产生负面影响？

- 新兴或实力更强的竞争对手
- 客户偏好或需求的变化
- 技术颠覆或产品过时风险
- 监管变化或合规风险
- 经济下行或市场萎缩
- 供应链中断
- 供应商或合作伙伴整合

## 输出流程
1. 识别 5-7 个优势（对竞争优势保持客观诚实）
2. 列出 5-7 个劣势（不要轻描淡写；聚焦可改进的差距）
3. 梳理 5-7 个机会（按市场规模和战略契合度排序）
4. 标注 5-7 个威胁（评估发生概率和影响程度）
5. 交叉分析，提炼战略洞察：
   - 如何借助优势抓住机会？
   - 如何补强劣势以抵御威胁？
   - 哪些机会能弥补劣势？
   - 哪些威胁可能被劣势放大？
6. 制定 3-5 条战略建议
7. 确定行动优先级和负责人
8. 识别追踪进度的指标

## 战略应用方向
- **进攻（Build）**：聚焦优势 + 机会的叠加点
- **防御（Defend）**：强化劣势 + 缓解威胁
- **转型（Pivot）**：探索能改变竞争格局的机会
- **退出（Exit）**：如果威胁过多且竞争地位较弱

## 备注
- SWOT 是从内部到外部的系统性评估
- 背景至关重要：与竞争对手和行业标准进行对比
- 每季度或市场条件发生重大变化时更新 SWOT
- 用 SWOT 指导产品路线图、合作策略和资源分配决策
- 机会和威胁应同时考虑当前动态和新兴动态
