# /market-scan -- 宏观环境分析

运行多种战略分析框架，全面了解竞争格局和宏观环境。将 SWOT、PESTLE、波特五力和安索夫矩阵整合为一份战略全景报告。

## 调用方式

```
/market-scan 企业学习领域的教育科技市场
/market-scan [上传市场简报或战略文档]
/market-scan 我们的金融科技产品——准备向董事会做战略汇报
```

## 工作流程

### 第一步：理解上下文

询问：
- 你要分析的是哪个产品、公司或市场？
- 分析目的是什么？（战略规划、市场进入、投资者准备、年度复盘）
- 是否有特别关注的框架？还是四个框架全部运行？
- 你目前在这个市场中处于什么位置？

### 第二步：运行分析

依次应用四种技能，每一种都在前一种洞察的基础上深入：

**SWOT 分析**（调用 **swot-analysis** 技能）：
- 内部：优势与劣势
- 外部：机会与威胁
- 各象限的可落地行动建议

**PESTLE 分析**（调用 **pestle-analysis** 技能）：
- 政治、经济、社会、技术、法律、环境六大因素
- 各因素的影响评估与时间线

**波特五力分析**（调用 **porters-five-forces** 技能）：
- 竞争对手、供应商、买方、替代品和新进入者威胁
- 行业整体吸引力评级

**安索夫增长矩阵**（调用 **ansoff-matrix** 技能）：
- 市场渗透、市场开发、产品开发、多元化
- 风险调整后的增长机会

### 第三步：综合分析

跨框架交叉比对，识别：
- **共同信号**：多个框架一致认同的结论
- **战略必做项**：跨分析均显示关键的行动
- **核心风险**：需要重点防范的威胁和力量
- **增长机会**：风险调整后最佳的增长路径

### 第四步：生成报告

```

**目的**：[战略规划 / 市场进入 / 等]
```

### 第五步：提供后续建议

- "要我基于这些发现**构建产品策略**吗？"
- "要我**深度分析波特五力中识别出的具体竞争对手**吗？"
- "要我为市场渗透机会**设计定价策略**吗？"

## 备注

- 使用网络调研将分析锚定于当前市场数据，而非仅凭通用知识
- PESTLE 各因素应包含具体法规、市场数据和趋势信号，而非泛泛而谈
- 波特五力最有价值之处在于识别*具体*的力量，而不只是做抽象评级
- 安索夫矩阵应包含具体的机会，而非"进入新市场"这类笼统表述
- 综合分析部分是整份报告最有价值的地方——各框架在这里相互印证、形成合力
