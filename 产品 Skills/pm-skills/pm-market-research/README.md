# pm-market-research

面向产品经理的市场调研技能包：用户画像、市场细分、情感分析与竞品分析。

## 技能列表（7 个）

- **competitor-analysis** — 分析竞品的优势、劣势与差异化机会。
- **customer-journey-map** — 创建端到端的客户旅程地图，涵盖阶段、触点、情绪、痛点与改进机会。
- **market-segments** — 识别 3-5 个潜在客户细分市场，包含人口统计、JTBD（待完成工作）与产品契合度分析。
- **market-sizing** — 使用自上而下与自下而上两种方法，估算 TAM（总体有效市场）、SAM（可服务市场）和 SOM（可获得市场）。
- **sentiment-analysis** — 分析用户反馈数据，识别各细分市场的情感评分、JTBD 与产品满意度洞察。
- **user-personas** — 基于调研数据构建精细化用户画像。
- **user-segmentation** — 从反馈数据中按行为、JTBD 和需求对用户进行细分。

## 命令列表（3 个）

- `/pm-market-research:analyze-feedback` — 大规模分析用户反馈——情感分析、主题提取与细分市场洞察。
- `/pm-market-research:competitive-analysis` — 分析竞争格局——识别竞品、对比优劣势、发现差异化机会。
- `/pm-market-research:research-users` — 全面的用户调研——基于调研数据构建用户画像、细分用户并绘制客户旅程地图。
