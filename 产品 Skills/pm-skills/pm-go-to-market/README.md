# pm-go-to-market

面向产品经理的市场进入技能包：GTM（市场进入策略）策略、增长循环、GTM 打法、桥头堡细分市场和理想客户画像。

## 技能列表（6 个）

- **beachhead-segment** — 确定产品上市的第一个桥头堡细分市场。
- **competitive-battlecard** — 制作可直接用于销售的竞争作战卡，全面对比你的产品与特定竞争对手。
- **growth-loops** — 识别可持续增长的增长循环（Growth Loops）。
- **gtm-motions** — 识别最适合的 GTM 打法和配套工具。
- **gtm-strategy** — 为产品上市制定完整的市场进入策略，涵盖营销渠道、核心信息、成功指标和上市计划。
- **ideal-customer-profile** — 从调研数据中提炼理想客户画像（ICP），包含用户画像、行为特征、JTBD（用户待办任务）和核心需求。

## 命令列表（3 个）

- `/pm-go-to-market:battlecard` — 制作可直接用于销售的竞争作战卡，涵盖产品定位、功能对比、异议处理和赢单策略。
- `/pm-go-to-market:growth-strategy` — 设计可持续的增长机制，包括面向产品驱动和销售驱动策略的增长循环与 GTM 打法。
- `/pm-go-to-market:plan-launch` — 制定完整的市场进入策略，包括桥头堡细分市场、ICP（理想客户画像）、核心信息、渠道规划和上市计划。
