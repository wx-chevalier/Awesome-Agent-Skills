## 竞争作战卡

为特定竞争对手制作一份简洁、可直接用于销售的竞争作战卡。

### 背景

你正在为 **$ARGUMENTS** 制作竞争作战卡。

使用网络搜索调研竞争对手的当前产品、定价、定位和近期动态。如果用户提供了文件（功能清单、赢单/败单数据、销售通话记录），请先阅读这些材料。

### 操作指引

1. **调研竞争对手**（使用网络搜索）：
   - 当前产品功能和产品线
   - 定价层级和模式
   - 目标市场和产品定位
   - 近期产品发布或功能变化
   - 已知优势和劣势
   - 用户评价和口碑（G2、Capterra、Reddit）

2. **按以下结构制作竞争作战卡**：

   ### 公司概况
   - 成立时间、总部、融资/营收（如为公开信息）
   - 目标市场和 ICP（理想客户画像）
   - 一句话定位

   ### 快速对比

   | 能力维度 | 我们 | 他们 | 优势方 |
   |---------|------|------|-------|
   | [功能领域 1] | [我们的方案] | [他们的方案] | [我们/他们/持平] |
   | [功能领域 2] | ... | ... | ... |
   | 定价 | ... | ... | ... |
   | 支持服务 | ... | ... | ... |

   ### 我们的优势所在
   - [优势 1]：[佐证材料或客户引言]
   - [优势 2]：[他们所不具备的具体能力]
   - [优势 3]：[更好的方案及理由]

   ### 他们的优势所在
   - [他们的强项 1]：[我们的对位策略]
   - [他们的强项 2]：[我们如何弥补这一差距]

   ### 常见异议与应对话术

   | 客户说 | 如何回应 |
   |--------|---------|
   | "竞争对手 X 有 [功能]" | "[我们的替代方案及为何更适合他们]" |
   | "他们更便宜" | "[价值框架：总拥有成本（TCO）、ROI、隐性成本]" |
   | "他们更成熟" | "[我们的优势：速度、创新、专注度、支持服务]" |

   ### 埋雷问题（Landmines）
   引导潜在客户主动暴露竞争对手弱点的问题：
   - "对你们团队而言，[我们擅长的领域] 有多重要？"
   - "你们评估过 [他们所缺乏的具体能力] 吗？"

   ### 赢单/败单规律
   - 我们倾向于赢单的情况：[规律]
   - 我们倾向于败单的情况：[规律]
   - 竞争商机中的关键差异化点：[能扭转局面的因素]

3. **保持一目了然**：销售代表需要在通话时随时查阅。请使用表格、加粗文字和简短的要点。

保存为 Markdown 格式。适配 Notion/Confluence 中的打印或分享需求。
