# /growth-strategy -- 增长循环与 GTM 打法

识别并设计驱动可持续增长的机制。评估五种增长循环类型和七种 GTM 打法，构建均衡的获客与扩张策略。

## 调用方式

```
/growth-strategy B2B 协作工具——增长已停滞在 5K 用户
/growth-strategy 面向消费者的健身 App，寻求病毒式增长
/growth-strategy [上传产品指标或增长数据]
```

## 工作流程

### 第一步：了解增长背景

询问以下问题：
- 产品是什么？目标用户是谁？
- 当前增长指标：用户数、增长率、获客渠道
- 什么在起作用？什么不奏效？
- 商业模式：收入与用户增长的关系是什么？
- 团队和预算：可以投入多少资源用于增长？

### 第二步：评估增长循环

调用 **growth-loops** 技能：

分析适用于你的产品的五种增长循环类型：

1. **病毒式飞轮（Viral Loop）**：用户在正常使用产品过程中自然邀请他人
2. **使用飞轮（Usage Loop）**：使用越多，价值越高，用户持续回归
3. **协作飞轮（Collaboration Loop）**：与他人共同使用时产品价值倍增
4. **用户生成内容飞轮（UGC Loop）**：用户创造内容，吸引新用户
5. **转介绍飞轮（Referral Loop）**：满意的用户主动向他人推荐

针对每个适用的飞轮，输出：运作机制、所需条件、预期影响、实施难度。

### 第三步：评估 GTM 打法

调用 **gtm-motions** 技能：

评估七种 GTM 打法：

1. **入站营销（Inbound）**：内容营销、SEO、思想领导力
2. **出站销售（Outbound）**：主动销售、冷邮件、基于账户的销售
3. **付费数字广告（Paid Digital）**：SEM（搜索引擎营销）、社交广告、展示广告、再营销
4. **社区营销（Community）**：论坛、活动、用户群、开发者关系
5. **合作伙伴营销（Partners）**：产品集成、经销商、联合营销
6. **ABM（基于账户的营销）**：精准企业获客
7. **PLG（产品驱动增长）**：免费版本、自助服务、产品病毒性

针对每种打法，输出：与你产品的匹配度、预期 CAC（客户获取成本）、见效时间、所需工具。

## 注意事项

- 增长循环会产生复利效应，而单次增长战术不会——优先构建飞轮，而非一次性活动
- 最好的增长循环是以产品本身为传播渠道（PLG、病毒式、协作式）
- 并非每种飞轮都适合每款产品——B2B 分析工具不会在 TikTok 上爆红
- 预算应跟着学习走：先小规模实验，验证有效后再加大投入
- CAC 应低于 LTV（用户生命周期价值）的 1/3，增长才可持续——若预期 CAC 过高需及时预警
