# pm-marketing-growth

产品营销与增长技能包：营销创意、价值主张陈述、北极星指标（North Star Metric）、产品命名与市场定位。

## 技能列表（5 个）

- **marketing-ideas** — 生成 5 个创意十足、性价比高的营销方案，包含渠道、信息传达角度与用户参与逻辑。
- **north-star-metric** — 确定北极星指标（North Star Metric）及 3-5 个输入指标。
- **positioning-ideas** — 头脑风暴产品定位方案，形成与竞争对手的差异化区隔。
- **product-name** — 头脑风暴 5 个独特易记的产品名称，附带命名理由，与品牌价值观及目标受众保持一致。
- **value-prop-statements** — 基于现有价值主张，生成面向营销、销售和用户引导场景的价值主张陈述。

## 命令列表（2 个）

- `/pm-marketing-growth:market-product` — 头脑风暴营销创意、定位方案、价值主张陈述与产品名称——一站式营销创意工具包。
- `/pm-marketing-growth:north-star` — 定义北极星指标（North Star Metric）及配套输入指标——识别商业模式类型，并对照最佳实践进行验证。
