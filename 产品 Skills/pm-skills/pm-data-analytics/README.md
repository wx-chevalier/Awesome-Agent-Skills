# pm-data-analytics

面向产品经理的数据分析技能包：SQL 查询生成与同期群分析。分析用户数据、生成查询语句，并识别留存规律。

## 技能列表（3 个）

- **ab-test-analysis** — 分析 A/B 测试结果，涵盖统计显著性检验、样本量验证、置信区间计算及可执行建议。
- **cohort-analysis** — 对用户参与度数据执行同期群分析。
- **sql-queries** — 将自然语言描述转化为 SQL 查询语句。

## 命令列表（3 个）

- `/pm-data-analytics:analyze-cohorts` — 对用户数据执行同期群分析——留存曲线、功能采用率及参与度趋势。
- `/pm-data-analytics:analyze-test` — 分析 A/B 测试结果——统计显著性、样本量验证，以及上线/延长/停止的决策建议。
- `/pm-data-analytics:write-query` — 用自然语言生成 SQL 查询——支持 BigQuery、PostgreSQL、MySQL 等多种方言。
